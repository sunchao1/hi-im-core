## Week 2 · Hub 核心路径

### 2.1 基础设施

- [ ] `src/hub/queue.hpp` — 有界 SPSC 环（先实现，后再优化）
- [ ] `src/hub/pipe_wakeup.hpp` — eventfd/pipe 唤醒
- [ ] `include/hiim/hub/context.hpp` — HubContext（connq/recvq/sendq/distq）

### 2.2 线程模块

- [ ] `src/hub/listener.cpp` — accept → connq
- [ ] `src/hub/reactor.cpp` — epoll 循环、收包、拼帧、写 socket
- [ ] `src/hub/distributor.cpp` — distq → sendq
- [ ] `src/hub/worker.cpp` — recvq → handler
- [ ] `src/hub/router.cpp` — SUB 表、nid→reactor 映射

### 2.3 连接协议

- [ ] AUTH 处理
- [ ] SUB / UNSUB（如需要）
- [ ] KPALIVE

### 2.4 bridge

- [ ] `src/hub/bridge.cpp`
  - [ ] FORWARD 上行业务 → `Publish(BACKEND, …)`
  - [ ] BACKEND 下行业务 → 读 IM 头 nid → `AsyncSend(FORWARD, …)`

### 2.5 主程序

- [ ] `cmd/hi-im-hub/main.cpp` — 双平面启动、信号 graceful shutdown

**Week 2 验收**：两个 `hi-im-proxy`（或最小 TCP 测试客户端）连 FORWARD/BACKEND，AUTH+SUB+AsyncSend 端到端通。
